# SuperYou Website

This repository hosts the SuperYou website on GitHub Pages.

## URLs

- **Main Site:** https://superyou.life
- **Privacy Policy:** https://superyou.life/privacy
- **Terms of Service:** https://superyou.life/terms

## Custom Domain Setup

To configure superyou.life with GitHub Pages:

1. Go to Repository Settings > Pages
2. Add custom domain: `superyou.life`
3. Enable HTTPS

### DNS Configuration

Add these DNS records for superyou.life:

| Type | Name | Value |
|------|------|-------|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |
| CNAME | www | funmania.github.io |
